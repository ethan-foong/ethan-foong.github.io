Multifunctional Robotic Manipulation

2025-09-05

Over the summer of 2025, I worked with the SLURM Lab at USC on a project investigating multifunctional robot manipulation. This slide shows our motivation and some of the ways we've thought of solving the problem.

This paper is now out! Check out HANDFUL!

And here are some of the things I've developed in this project.

A robot hand teleoperation system that can help test manipulation task feasability and allow collection of manipulation demonstrations for robot learning. Works with xArm7, Allegro and Leap hands, and the Rokoko motion capture Gloves.

A reinforcement learning pipeline that will allow dexterous robot hands to complete multi-functional tasks (like grabbing an object while pushing another, grasping an object and opening a drawer etc.) Simulations done in Maniskill and Isaac Gym.
